# card-hover-effect
A simple card hover effect using HTML &amp; CSS, featuring layered stacking with z-index and rotation effects.

## Features
- Uses `z-index` to change stacking order.
- CSS `rotate` effect for a stylish look.
- Fully responsive.

## Technologies Used
- HTML
- CSS
